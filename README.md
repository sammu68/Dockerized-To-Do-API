This project is a Dockerized To-Do API built using Python and FastAPI, designed to demonstrate basic backend development and containerization concepts. The application provides RESTful endpoints to create, read, update, and delete tasks, with data stored in a lightweight SQLite database. It is fully containerized using Docker, allowing the application to run consistently across different environments without additional setup. The project highlights key concepts such as API development, database integration, and Docker-based deployment, making it a beginner-friendly yet practical example for learning modern backend workflows and showcasing skills on GitHub.
